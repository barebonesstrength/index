Settings

Configure your protocol and manage your data.

Protocol start date
Set this to the first day of your current protocol. All chart labels switch from "W1, W2..." to real calendar dates like "6 Jan", "13 Jan".
Training phases
The four default phases are always available. Add custom phases for your specific protocol — like Peaking, Reverse Diet, or Maintenance Surplus. Custom phases can be removed at any time.
Export & Import
Export a JSON backup of all your data. Import a previous export to restore your history on a new device or browser. Your data saves automatically in this browser.
Sample data
Load a 12-week cut phase example to explore all features with realistic data. This will replace any existing data.
Clear all data
Permanently delete all weeks and daily logs from this browser. Export first if you want a backup — this cannot be undone.
